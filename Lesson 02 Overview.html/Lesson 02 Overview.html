CIT 365 - .NET Software Development using C#

Lesson 02 Overview

This week you will continue to leverage your knowledge about common control structures and programming concepts with the C# programming language. We will investigate selection and iteration structures, operators, statements, methods, arrays, exceptions, and the switch structure. By the end of the week, you will be able to converse and work with others using these concepts and build a basic desktop application. 

Objectives

  • Compare common programming concepts from prerequisite knowledge to the C# language
  • Identify output given code snippets from the concepts of the chapters
  • Debug erroneous C# code segments given stated functional specifications
  • Establish a version control, collaboration pattern using a remote repository, GitHub
  • Build a desktop application with given well-defined, functional specifications
  • Establish a pattern of peer review and collaboration in an online setting
